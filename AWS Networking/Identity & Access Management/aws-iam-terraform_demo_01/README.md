# aws-iam-terraform_demo_01

This folder contains AWS IAM (Identity and Access Management) Terraform demonstration code.

## Overview

This Terraform project demonstrates how to manage AWS IAM resources including users, roles, groups, and policies.

## Files

- `provider.tf` - AWS provider configuration
- `users.tf` - IAM user definitions
- `roles.tf` - IAM role definitions
- `groups.tf` - IAM group definitions
- `policies.tf` - IAM policy definitions
- `variables.tf` - Terraform variables
- `outputs.tf` - Terraform outputs

## Getting Started

1. Ensure Terraform is installed
2. Configure AWS credentials
3. Initialize Terraform: `terraform init`
4. Review changes: `terraform plan`
5. Apply changes: `terraform apply`
